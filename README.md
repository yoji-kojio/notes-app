# notes-app

notes-app is an application to create and manage your notes.

## Installation
```bash
npm install
```

## Usage

### Adding a note

```bash
node app.js add --title="My note" --body="My note body!"
```

### Removing a note

```bash
node app.js remove --title="My note"
```

### List all notes

```bash
node app.js list
```

### Reading a note

```bash
node app.js read --title="My note"
```
